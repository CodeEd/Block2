Review of HTML tags and attributes from Block 1

Code What it means
<html> ... <html> Start web page and end web page
<head> ... <head> Where information goes but is not inside the window
<title> ... <title> The name that appears on the browser tab/bar
<body> ... <body> Where everything seen in the browser window goes
<b> ... <> Bold text
<i> ... <i> Italic text
<u> ... <u> Underlined text
<h1> ... <h1> Biggest header
<h2> ... <h2> Second biggest header
<p> ... <p> Paragraph
<a href="..."> ... <a> Link
<img src="..." />
<div> ... <div> Container for block elements
<span> ... <span> Container for text
<ol> ... <ol> Ordered list
<ul> ... <ul> Unordered list
<li> ... <li> List item

Creating, opening, saving, and compiling

Command What it does
CTRL + N Creates a new file/web page
CTRL + O Opens a file/web page
CTRL + S Saves the current file/web page
Compile Saves all files/web pages to online server
